## Trabajos realizados y subidos

- AlcanzAR es una aplicación móvil de carpooling dinámico diseñada para conectar personas que realizan trayectos similares, ya sean viajes diarios de trabajo/estudio o traslados de larga distancia entre distintas localidades.

- El mismo objetivo de la aplicación lleva a pensar que su utilización puede ir más alla de viajes. Un usuario podría utilizar tranquilamente la aplicación para solicitar herramientas/objetos. De allí su nombre "AlcanzAR". 

- La aplicación obtiene gran valor al ser móvil, ya que depende del GPS para validar la ubicación del usuario, mejorar la seguridad y ofrecer coincidencias de viaje en tiempo real. Además, utiliza notificaciones para avisar cuando aparece una oportunidad de viaje o cuando otro usuario responde a una solicitud.
